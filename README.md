# Interview Simulator

## Overview

An adaptive interview simulator.

Questions become harder if the candidate performs well.

---

## Files

- interview_engine.py
- simulator.py

---

## Run

```bash
python simulator.py
```

---

## Features

- Easy Questions
- Medium Questions
- Hard Questions
- Automatic Scoring
- Adaptive Difficulty

---

## Example

Question:

What is Python?

Answer:

Python is a programming language.

Points Earned: 2

---

Question:

Explain OOP concepts.

Answer:

Inheritance, polymorphism and encapsulation.

Points Earned: 3

---

Final Score: 5
